## ⚠️ 현재 상황

## 🎯 방법 1: GitHub 웹사이트에서 직접 업로드 (가장 쉬움!)

### 1단계: GitHub 리포지토리 접속
1. 브라우저를 열고 다음 주소로 이동:
   ```
   https://github.com/mathservice004-collab/tetris
   ```
2. GitHub에 로그인

### 2단계: 파일 업로드
1. **"Add file"** 버튼 클릭 → **"Upload files"** 선택
2. 파일 탐색기에서 `c:\dev\test01` 폴더 열기
3. **다음 파일들을 모두 선택**하여 드래그 & 드롭:

   #### 📄 루트 파일들:
   - `index.html`
   - `README.md`
   - `QUICKSTART.md`
   - `LICENSE`
   - `.gitignore`

   #### 📁 폴더들:
   - `styles` 폴더 전체
   - `src` 폴더 전체

4. 브라우저 창으로 드래그 & 드롭 (또는 "choose your files" 클릭)

### 3단계: 커밋 메시지 작성
페이지 하단의 커밋 섹션에:

**Commit message:**
```
Initial commit: TETRIS: REFINED
```

**Extended description (선택사항):**
```
✨ Features:
- Classic tetris with modern UI/UX
- SRS rotation system with wall kicks
- 7-bag randomizer for fair piece distribution
- 60fps performance with <1 frame input latency
- DAS/ARR input system
- Hold and Next Queue (5 pieces preview)
- Ghost piece
- Back-to-Back Tetris bonus
- High score persistence

🎮 Play instantly by opening index.html!
```

### 4단계: 업로드 완료
1. **"Commit changes"** 버튼 클릭
2. 업로드 완료! 🎉

## 🎁 보너스: GitHub Pages로 온라인 플레이 가능하게 만들기

업로드가 완료되면:

### 1단계: Settings로 이동
1. 리포지토리 페이지에서 **"Settings"** 탭 클릭

### 2단계: Pages 설정
1. 왼쪽 메뉴에서 **"Pages"** 선택
2. **Source** 섹션:
   - Branch: `main` 선택
   - Folder: `/ (root)` 선택
3. **"Save"** 클릭

### 3단계: URL 확인
몇 분 후 다음 주소에서 게임을 온라인으로 플레이할 수 있습니다:
```
https://mathservice004-collab.github.io/tetris/
```

## 🔧 방법 2: Git 설치 후 자동 업로드 (더 고급)

만약 Git을 사용하고 싶다면:

### Git 설치
1. https://git-scm.com/download/win 에서 다운로드
2. 기본 설정으로 설치
3. PowerShell을 **새로 열기** (중요!)

### 자동 업로드 스크립트
PowerShell에서 실행:

```powershell
cd c:\dev\test01
git init
git add .
git commit -m "Initial commit: TETRIS: REFINED"
git remote add origin https://github.com/mathservice004-collab/tetris.git
git branch -M main
git push -u origin main
```

인증이 필요하면:
- Username: GitHub 사용자명
- Password: Personal Access Token (비밀번호 아님!)
  - GitHub → Settings → Developer settings → Personal access tokens → Generate new token
  - `repo` 권한 체크
