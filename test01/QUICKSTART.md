# TETRIS: REFINED - 실행 가이드

## 🚀 빠른 시작

TETRIS: REFINED를 로컬에서 실행하는 방법입니다.

### 방법 1: 파일 더블클릭 (가장 간단!)

1. 파일 탐색기에서 `c:\dev\test01` 폴더 열기
2. `index.html` 파일을 더블클릭
3. 기본 브라우저에서 게임이 자동으로 열립니다!

### 방법 2: 드래그 & 드롭

1. `index.html` 파일을 선택
2. 브라우저 창으로 드래그 & 드롭
3. 게임 실행!

### 방법 3: 브라우저에서 직접 열기

브라우저 주소창에 다음을 입력:
```
file:///c:/dev/test01/index.html
```

## 🎮 게임 시작하기

1. 브라우저에서 게임이 열리면 "START GAME" 버튼 클릭
2. 조작법:
   - **← →** : 좌우 이동
   - **↓** : 빠른 낙하
   - **SPACE** : 즉시 낙하
   - **Z / X** : 회전
   - **C** : 조각 보관 (Hold)
   - **P / ESC** : 일시정지

## ✅ 구현 완료된 기능

### 핵심 게임플레이
- ✅ 7가지 테트로미노 (I, O, T, S, Z, J, L)
- ✅ 7-Bag 랜덤 시스템 (공정한 조각 분배)
- ✅ SRS 회전 시스템 (Wall Kick 지원)
- ✅ Hold 기능
- ✅ Next Queue (다음 5개 미리보기)
- ✅ Ghost Piece (착지 위치 표시)

### 점수 시스템
- ✅ Single/Double/Triple/Tetris 점수 차등
- ✅ Back-to-Back Tetris 보너스 (50% 추가)
- ✅ Soft Drop / Hard Drop 점수
- ✅ 레벨별 점수 배수
- ✅ 10라인마다 레벨 업
- ✅ 최고 점수 저장

### 입력 시스템

- ✅ ARR (Auto Repeat Rate) - 2프레임
- ✅ Lock Delay - 30프레임
- ✅ 프레임 기반 입력 처리

### 퍼포먼스
- ✅ 60fps 고정 프레임레이트
- ✅ Canvas 기반 최적화 렌더링
- ✅ 최소 입력 지연

### UI/UX
